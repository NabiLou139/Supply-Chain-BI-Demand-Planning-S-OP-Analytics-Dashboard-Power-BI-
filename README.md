# Supply-Chain-BI-Demand-Planning-S-OP-Analytics-Dashboard-Power-BI-
End-to-end Supply Chain BI project built with Power BI for Demand Planning and S&amp;OP. Integrates Production, Inventory, Sales, BOM, and S&amp;OP tables to provide automatic insights on production rates, lead times, inventory evolution, sales analysis, forecasts, remaining production, and raw material needs.
